  • Website Hosting

    What is Website Hosting?

    When you decide to start a website, you need to find website hosting. The hosting company will provide a url for users to reach your website (Ex, www.google.com). I’ll narrow down the options into four types of hosting: shared hosting, VPS hosting, cloud hosting, and dedicated hosting.

  • Shared Hosting

    What is Shared Hosting?

    If you decided yo use shared hosting your website will share the same resources, such as CPU, disk space, and memory with other users who are on the same shared hosting server. You can illustrate Shared Hosting as living in an apartment where you have to share space.

    Shared hosting is the lowest price when comparing the options and is great to use for low traffic, starter websites. Unfortunately, your site might have limited bandwidth and will get slow when there is a lot of traffic crowding website’s on the same server as you are.

  • VPS Hosting

    What is VPS Hosting?

    With Virtual Private Servers you’ll still share your server, but each user gets dedicated resources – meaning other VPS on the same physical server won’t affect you. Still using an analogy, a VPS partition is like having a penthouse, or a condo.

    VPS hosting provides a dedicated virtual partition for each user, making sure resources are always available to you. Yet, of course, VPS comes at a higher price.

  • Cloud Hosting

    What is Cloud Hosting?

    Cloud hosting is a type of web hosting which uses multiple servers to balance the traffic and maximize uptime. It is inspired by cloud computing technologies that allow a large number of machines to work as one system.

    Instead of having your site hosted in one server, it’s in a cluster which allows multiple servers to work together, improving redundancy and traffic management. If one server fails, another will aid in keeping everything running.

  • Dedicated Hosting

    What is Dedicated Hosting?

    If you are using a dedicated hosting service, your website will be stored in a single server dedicated only for your personal use. You don’t have to share the same server with anyone, and you can freely customize it based on your liking.

    A dedicated server is often the most expensive option but a great choice for large scale applications.
